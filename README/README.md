Um site para fazer listas de supermercado.
Funcionalidades.
•	Cadastro de usuário
•	Login de usuário
•	Adicionar itens
•	Cadastro da quantidade e valor do produto
•	Remover itens
•	Limpar a lista toda do supermercado
•	Modo claro e modo escuro

O que fazer para melhoria.
